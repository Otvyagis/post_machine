# Машина Поста
## Описание
Данный проект представляет собой программную реализацию Машины Поста — абстрактной вычислительной машины, предложенной Эмилем Постом. Это простейшая модель вычислений, эквивалентная по вычислительной мощности машине Тьюринга.
Приложение предоставляет графический интерфейс для создания, редактирования и выполнения программ для Машины Поста, с возможностью пошагового выполнения и визуализации состояния ленты.
##Особенности
 - Графический интерфейс на основе Tkinter
 - Визуализация ленты с текущей позицией считывающей головки
 - Редактор программ
 - Сохранение и загрузка программ в базу данных SQLite
 - История выполненных команд
 - Экспорт/импорт программ в текстовом формате
##Требования к системе
 - Python 3.8 или выше
 - SQLite3 (обычно входит в состав Python)
 - Tkinter (для графического интерфейса)
## Запуск
Установка зависимостей
``
pip install -r requirements.txt
``

Запуск
``
python main.py
``

## Docker
Собрать Docker:
``
docker build -t post-machine .
``

Запуск:
``
docker run -it --rm \
  -e DISPLAY=$DISPLAY \
  -v /tmp/.X11-unix:/tmp/.X11-unix \
  post-machine
``

## Использование

Команда: Описание

MARK: Записывает 1 в текущую ячейку ленты

ERASE: Записывает 0 в текущую ячейку ленты

LEFT: Сдвигает головку влево на одну позицию

RIGHT: Сдвигает головку вправо на одну позицию

IF1: метка	Переход к метке, если в текущей ячейке 1

IF0: метка	Переход к метке, если в текущей ячейке 0

GOTO: метка	Безусловный переход к метке
HALT: Остановка выполнения программы
